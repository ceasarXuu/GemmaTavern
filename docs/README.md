# Public Docs

This directory is reserved for stable, public-facing engineering references.

Keep here:

- architecture references,
- compatibility notes,
- repeatable validation standards,
- benchmark automation that another contributor can run.

Do not keep here:

- internal roadmaps,
- one-off engineering notes,
- temporary migration checklists,
- generated reports,
- presentation or pitch material.
